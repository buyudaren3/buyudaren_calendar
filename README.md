# 捕鱼达人日历

自动生成包含中国农历节日的 ICS 日历文件。

## 订阅地址

推荐使用 GitHub Pages 地址（国内可直接访问）：

```
https://buyudaren3.github.io/buyudaren_calendar/buyudaren_calendar.ics
```

备用地址（需要代理）：

```
https://raw.githubusercontent.com/buyudaren3/buyudaren_calendar/main/buyudaren_calendar.ics
```

## iPhone 订阅步骤

1. 设置 → 日历 → 账户 → 添加账户
2. 选择「其他」→「添加已订阅的日历」
3. 粘贴上面的 URL
4. 点击「下一步」→「存储」

## 包含的节日

### 中国法定及传统节日（自动重复）
- 元旦、妇女节、植树节、劳动节、青年节
- 儿童节、建党节、建军节、教师节、国庆节

### 节气（逐年计算）
- 清明节（4月4日或5日，每年自动计算）

### 其他公历节日（自动重复）
- 情人节、愚人节、万圣夜、万圣节、平安夜、圣诞节
- 母亲节、父亲节、感恩节

### 农历节日（逐年计算）
- 腊八节、小年（南北）、除夕、春节
- 元宵节、龙抬头、端午节、七夕
- 中元节、中秋节、重阳节

## 自动更新

GitHub Actions 会在每月1日自动运行脚本更新日历。
