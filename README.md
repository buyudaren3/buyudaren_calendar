# 捕鱼达人日历

自动生成包含中国农历节日的 ICS 日历文件。

## 订阅地址

将此仓库推送到 GitHub 后，在 iPhone 上订阅：

```
https://raw.githubusercontent.com/buyudaren3/buyudaren_calendar/main/buyudaren_calendar.ics
```

## iPhone 订阅步骤

1. 设置 → 日历 → 账户 → 添加账户
2. 选择「其他」→「添加已订阅的日历」
3. 粘贴上面的 URL
4. 点击「下一步」→「存储」

## 包含的节日

### 公历节日（自动重复）
- 情人节、植树节、愚人节、教师节
- 万圣夜、万圣节、平安夜、圣诞节
- 母亲节、父亲节、感恩节

### 农历节日（自动计算）
- 腊八节、小年（南北）、除夕、春节
- 元宵节、龙抬头、端午节、七夕
- 中元节、中秋节、重阳节

## 自动更新

GitHub Actions 会在每年 1月1日 和 7月1日 自动运行脚本更新日历。
